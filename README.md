# 📧 Email Automation & Reminder System

## 🚀 Project Overview
This project automates email reminders using Python. It reads contacts from CSV, schedules reminders, sends emails (simulated), and generates reports with graphs.

---

## 📌 Features
- CSV-based contact management
- Email reminder automation
- Scheduling system
- Logging system
- Report generation
- Data visualization (graphs)

---

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- CSV
- Google Colab

---

## 📊 Output Example
- Email status report
- Success vs failure graph
- Automated logs

---

## 📈 Graphs Included
- Email Status Bar Chart
- Email Success Rate Pie Chart

## 📂 Files
